# Text-Based RPG Game (C++)

## Overview
A text-based role-playing game developed in C++ that allows players to progress through an interactive storyline using decision-based gameplay. The project focuses on game logic, player progression, and system design using object-oriented programming principles.

## Gameplay
- Players navigate the game through text-based choices  
- Decisions impact outcomes, progression, and interactions  
- Includes combat encounters, exploration, and player actions  
- Game state updates dynamically based on user input  

## Features
- Interactive decision-based gameplay system  
- Player progression and game state tracking  
- Combat mechanics and enemy encounters  
- Input-driven command system  
- Modular code structure using object-oriented design  

## Technologies
- C++  
- Object-Oriented Programming (OOP)  

## Project Structure
src/        # Source files (.cpp)  
include/    # Header files (.h)  

## How to Run
1. Clone the repository:
   git clone https://github.com/BasharMakani/TextBasedRpg.git  

2. Navigate to the project directory  

3. Compile the program:
   g++ main.cpp -o rpg  

4. Run the executable:
   ./rpg  

## Example Gameplay
- Choose actions through terminal input  
- Engage in combat or exploration  
- Progress through different scenarios based on decisions
